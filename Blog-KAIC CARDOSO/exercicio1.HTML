Web Semântica

Evoluindo a já classica e conhecida "Web de documentos" o WC3 no
desenvolvimento de tecnologias que darão suporte a "Web dos dados"
viabilizando pesquisas como num banco de dados.
O objetivo final da Web de dados é possibilitar que computadores façam
coisas mais úteis e com que o desenvolvimento de sistemas possa oferecer
suporte a interações na rede. O termo"Web Semântica" refere-se á visão do
WC3 da Web dos Dados Linkados.
A Web Semântica dá ás pessoas a capacidade de criarem repositórios de
dados na Web. construirem vocabulários e escreverem regras para
interoperarem com esses dados.
